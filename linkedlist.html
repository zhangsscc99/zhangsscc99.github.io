力扣328 奇偶链表

            class Solution {
                public ListNode oddEvenList(ListNode head) {
                    if (head == null) {
                        return head;
                    }
                    ListNode even = head.next;
                    ListNode odd = head;
                    ListNode evenHead = head.next;
                    while (even != null && even.next != null) {
                        odd.next = even.next;
                        odd = odd.next;
            
                        even.next = odd.next;
                        even = even.next;
                    }
                    odd.next = evenHead;
                    return head;
            
            
                }
            }

这道题关键点在于 设置好odd和even的起始点。 odd.next = even.next; odd = odd.next; 先把指针处理好，再处理节点递进。 while (even != null && even.next != null) 常用模板。 最后用前面一个接上后面一个就可以了。

876. 链表的中间结点

            class Solution {
                public ListNode middleNode(ListNode head) {
                    ListNode fast = head;
                    ListNode slow = head;
                    
            
                    while (fast != null && fast.next != null) {
                        fast = fast.next.next;
                        slow = slow.next;
                    }
            
                    return slow;
            
                }
            }

这道题关键点在于 设置快慢指针。数一数，当fast没法继续前进的时候，slow正好是中间节点。。

160. 相交链表

            public class Solution {
                public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
                    ListNode p1 = headA;
                    ListNode p2 = headB;
            
                    while (p1 != p2) {
                        if (p1 != null) {
                            p1 = p1.next;
                        } else {
                            p1 = headB;
                        }
            
                        if (p2 != null) {
                            p2 = p2.next;
                        } else {
                            p2 = headA;
                        }
                    }
            
                    return p1;
                    
                }
            }

这道题关键点在于 p1 p2 要能够等于null。因为有不存在相交节点的case。所以需要用等于null的情况来返回正确的返回值。不能写if (p1.next != null)

206 反转链表

                class Solution {
                    public ListNode reverseList(ListNode head) {
                        if (head == null || head.next == null) {
                            return head;
                        }
                
                        ListNode last = reverseList(head.next);
                        head.next.next = head;
                        head.next = null;
                
                        return last;
                
                    }
                }

这道题关键点在于用递归。

2 两数相加

        class Solution {
            public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
                ListNode dummy = new ListNode(0);
                ListNode cur = dummy;
                int carry = 0;
        
                while (l1 != null || l2 != null || carry != 0) {
                    if (l1 != null ) {
                        carry += l1.val;
                        l1 = l1.next;
                    }
        
                    if (l2 != null) {
                        carry += l2.val;
                        l2 = l2.next;
                    }
        
                    cur.next = new ListNode(carry % 10);
                    cur = cur.next;
                    carry = (int)(carry / 10);
        
                }
        
                return dummy.next;
        
        
        
            }
        }

这道题关键点在于正好可以用逆序进行答案的返回，所以正好可以把进位往next接。cur是需要不断取余数的 而且cur需要把进位进行更新 while里的条件也至关重要 里面的carry l1 l2任意一个不为0或者null的时候 都是可以往前进的。int要用整除。

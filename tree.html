力扣111 二叉树的最小深度

            /**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public int minDepth(TreeNode root) {
        if (root == null) {
            return 0;
        }

        Queue q = new LinkedList<>();
        q.offer(root);
        int depth = 0;

        while (!q.isEmpty()) {
            int sz = q.size();
            for (int i = 0; i < sz; i++) {
                TreeNode cur = q.poll();
                if (cur.left == null && cur.right == null) {
                    depth++;
                    return depth;
                }

                if (cur.left != null) {
                    q.offer(cur.left);
                }

                if (cur.right != null) {
                    q.offer(cur.right);
                }

            

            }
            depth++;
            
            

        }

        return depth;

    }
}

这道题关键点在于 用层序遍历 模板要背熟 我很熟 一直都能熟练写对。 注意一开始的special case要好好处理。 while里面是对queue的size进行for循环 判断没有左右子节点就返回depth。

103 二叉树的锯齿形层序遍历

            class Solution {
                public List> zigzagLevelOrder(TreeNode root) {
                    if (root == null) {
                        LinkedList> empty = new LinkedList<>();
                        return empty;
                    }
            
                    LinkedList> res = new LinkedList<>();
                    Queue q = new LinkedList<>();
                    q.offer(root);
            
                    int depth = 1;
            
                    while (!q.isEmpty()) {
                        int sz = q.size();
                        LinkedList level = new LinkedList<>();
                        for (int i = 0; i < sz; i++) {
                            TreeNode cur = q.poll();
                            if (depth % 2 == 1) {
                                level.addLast(cur.val);
                            } else {
                                level.addFirst(cur.val);
                            }
            
                            if (cur.left != null) {
                                q.offer(cur.left);
                            }
                            if (cur.right != null) {
                                q.offer(cur.right);
                            }
                        }
                        depth++;
                        res.addLast(level);
                    }
            
                    return res;
            
            
                }
            }

我巧妙地利用了depth记录奇数层偶数层，再利用java linkedlist的addFirst和addLast功能，完美实现了这道题的解法。

101 对称的树

            # Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        if root == None:
            return False

        return self.dfs(root.left, root.right)
    def dfs(self, left, right):
        if (left == None and right == None):
            return True
        if (left == None or right == None):
            return False
        if (left.val != right.val):
            return False
        return self.dfs(left.left, right.right) and self.dfs(left.right, right.left)

主要是递归。用bfs做出来。都是null的时候就返回true，一个null就是false，都不null但是不相等也是false。想着什么样会是false会更容易一些。

2 两数相加

        class Solution {
            public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
                ListNode dummy = new ListNode(0);
                ListNode cur = dummy;
                int carry = 0;
        
                while (l1 != null || l2 != null || carry != 0) {
                    if (l1 != null ) {
                        carry += l1.val;
                        l1 = l1.next;
                    }
        
                    if (l2 != null) {
                        carry += l2.val;
                        l2 = l2.next;
                    }
        
                    cur.next = new ListNode(carry % 10);
                    cur = cur.next;
                    carry = (int)(carry / 10);
        
                }
        
                return dummy.next;
        
        
        
            }
        }

这道题关键点在于正好可以用逆序进行答案的返回，所以正好可以把进位往next接。cur是需要不断取余数的 而且cur需要把进位进行更新 while里的条件也至关重要 里面的carry l1 l2任意一个不为0或者null的时候 都是可以往前进的。int要用整除。

92 反转链表II

            class Solution {
                public ListNode reverseBetween(ListNode head, int left, int right) {
                    if (left == 1) {
                        return reverseN(head, right);
                    }
            
                    head.next = reverseBetween(head.next, left - 1, right - 1);
                    return head;
            
                }
            
                ListNode successor = new ListNode(0);
            
                public ListNode reverseN(ListNode head, int n) {
                    if (n == 1) {
                        successor = head.next;
                        return head;
            
                    }
            
                    ListNode last = reverseN(head.next, n - 1);
            
                    head.next.next = head;
                    head.next = successor;
            
                    return last;
            
            
                }
            }

这道题关键点在于需要熟练反转链表I。先把需要反转的区域调整到left == 1的位置。然后对剩下的右边的链表进行处理 注意处理successor。

142 环形链表II

        public class Solution {
            public ListNode detectCycle(ListNode head) {
                ListNode fast = head;
                ListNode slow = head;
        
                while (fast != null && fast.next != null) {
                    fast = fast.next.next;
                    slow = slow.next;
                    if (slow == fast) {
                        break;
                    }
                }
        
                if (fast == null || fast.next == null) {
                    return null;
                }
        
                fast = head;
        
                while (fast != slow) {
                    fast = fast.next;
                    slow = slow.next;
                }
        
                return slow;
                
            }
        }

先找到环里相交的点。然后此时把fast重新设置回head。注意 fast行走距离2k，slow行走距离k，假设交点距离环的起点距离m，那么head距离环的起点就是k - m。 此刻，如果fast从头开始到环的起点，slow也同步从交点开始行走，那么他们在环的起点可以相遇而且正好走了k-m距离。这个可以用作结论记住。

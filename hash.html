力扣2215 找出两数组的不同

            class Solution {
                public List> findDifference(int[] nums1, int[] nums2) {
                    List> res = new ArrayList<>();
                    res.add(new ArrayList<>());
                    res.add(new ArrayList<>());
                    Set s1 = new HashSet<>();
                    for (int i = 0; i < nums1.length; i++) {
                        s1.add(nums1[i]);
                    }
                    Set s2 = new HashSet<>();
                    for (int i = 0; i < nums2.length; i++) {
                        s2.add(nums2[i]);
                    }
            
                    for (Integer int1: s1) {
                        if (!s2.contains(int1)) {
                            res.get(0).add(int1);
                        }
                    }
            
                    for (Integer int2: s2) {
                        if (!s1.contains(int2)) {
                            res.get(1).add(int2);
                        }
                    }
            
                    return res;
            
            
                }
            }

看起来简单的哈希题都不要大意，这道题关键点在于 熟练地使用HashSet和List。这里res就用ArrayList是最方便的。为了避免重复元素出现在答案里，被遍历的数据结构就应该是哈希Set。

1 两数之和

            class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        keys = {}
        for i in range(len(nums)):
            if target - nums[i] in keys:
                return [keys[target - nums[i]], i]
            else:
                keys[nums[i]] = i

这道题关键点在于 设置快慢指针。数一数，当fast没法继续前进的时候，slow正好是中间节点。。

128 最长连续序列

            class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        set_num = set(nums)
        length = 0
        res = 0

        for i in range(len(nums)):
            if nums[i] - 1 in set_num:
                continue

            cur = nums[i]
            length = 1

            while cur + 1 in set_num:
                cur = cur + 1
                length = length + 1
            
            res = max(length, res)
        return res

这题如果用O(n) 说实话很难。实际做法是一个摊销分析后的O(n)。（amortized analysis）

206 反转链表

                class Solution {
                    public ListNode reverseList(ListNode head) {
                        if (head == null || head.next == null) {
                            return head;
                        }
                
                        ListNode last = reverseList(head.next);
                        head.next.next = head;
                        head.next = null;
                
                        return last;
                
                    }
                }

这道题关键点在于用递归。

2 两数相加

        class Solution {
            public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
                ListNode dummy = new ListNode(0);
                ListNode cur = dummy;
                int carry = 0;
        
                while (l1 != null || l2 != null || carry != 0) {
                    if (l1 != null ) {
                        carry += l1.val;
                        l1 = l1.next;
                    }
        
                    if (l2 != null) {
                        carry += l2.val;
                        l2 = l2.next;
                    }
        
                    cur.next = new ListNode(carry % 10);
                    cur = cur.next;
                    carry = (int)(carry / 10);
        
                }
        
                return dummy.next;
        
        
        
            }
        }

这道题关键点在于正好可以用逆序进行答案的返回，所以正好可以把进位往next接。cur是需要不断取余数的 而且cur需要把进位进行更新 while里的条件也至关重要 里面的carry l1 l2任意一个不为0或者null的时候 都是可以往前进的。int要用整除。

92 反转链表II

            class Solution {
                public ListNode reverseBetween(ListNode head, int left, int right) {
                    if (left == 1) {
                        return reverseN(head, right);
                    }
            
                    head.next = reverseBetween(head.next, left - 1, right - 1);
                    return head;
            
                }
            
                ListNode successor = new ListNode(0);
            
                public ListNode reverseN(ListNode head, int n) {
                    if (n == 1) {
                        successor = head.next;
                        return head;
            
                    }
            
                    ListNode last = reverseN(head.next, n - 1);
            
                    head.next.next = head;
                    head.next = successor;
            
                    return last;
            
            
                }
            }

这道题关键点在于需要熟练反转链表I。先把需要反转的区域调整到left == 1的位置。然后对剩下的右边的链表进行处理 注意处理successor。

141 环形链表

                public class Solution {
                    public boolean hasCycle(ListNode head) {
                        ListNode fast = head;
                        ListNode slow = head;
                
                        while (fast != null && fast.next != null) {
                            fast = fast.next.next;
                            slow = slow.next;
                            if (fast == slow) {
                                return true;
                            }
                
                        }
                        return false;
                    }
                }

快慢指针的while循环条件一般都是这个。

142 环形链表II

        public class Solution {
            public ListNode detectCycle(ListNode head) {
                ListNode fast = head;
                ListNode slow = head;
        
                while (fast != null && fast.next != null) {
                    fast = fast.next.next;
                    slow = slow.next;
                    if (slow == fast) {
                        break;
                    }
                }
        
                if (fast == null || fast.next == null) {
                    return null;
                }
        
                fast = head;
        
                while (fast != slow) {
                    fast = fast.next;
                    slow = slow.next;
                }
        
                return slow;
                
            }
        }

先找到环里相交的点。然后此时把fast重新设置回head。注意 fast行走距离2k，slow行走距离k，假设交点距离环的起点距离m，那么head距离环的起点就是k - m。 此刻，如果fast从头开始到环的起点，slow也同步从交点开始行走，那么他们在环的起点可以相遇而且正好走了k-m距离。这个可以用作结论记住。

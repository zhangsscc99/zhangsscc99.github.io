            class Solution {
                public String removeStars(String s) {
                    Stack stk = new Stack<>();
                    for (int i = 0; i < s.length(); i++) {
                        Character ch = s.charAt(i);
                        if (ch == '*') {
                            stk.pop();
                        } else {
                            stk.push(ch);
                        }
                    }
            
                    StringBuilder sb = new StringBuilder();
                    for (Character ch: stk) {
                        sb.append(ch);
                    }
                    return sb.toString();
            
                }
            }
        

        
这道题很简单，别犯傻就行了。
        


            

        
735 小行星碰撞

        
            class Solution:
    def asteroidCollision(self, asteroids: List[int]) -> List[int]:
        stk = []
        for aster in asteroids:
            alive = True
            while alive and aster < 0 and stk and stk[-1] > 0:
                if stk[-1] < -aster:
                    alive = True
                else:
                    alive = False
                if stk[-1] <= -aster:
                    stk.pop()
            if alive:
                stk.append(aster)
        return stk

        

        
注意 其实往左的行星是不需要去持续关注的。只需要关注碰撞的情况就可以了。是可以简化的。


            

        
394 字符串解码

        
            class Solution:
    def decodeString(self, s: str) -> str:
        stack = []
        res = ""
        multi = 0

        for c in s:
            if c == '[':
                stack.append([multi, res])
                res, multi = "", 0
            elif c == ']':
                cur_multi, last_res = stack.pop()
                res = last_res + cur_multi * res
            elif '0' <= c <= '9':
                multi = multi * 10 + int(c)
            else:
                res += c
        return res

2 两数相加

    
        class Solution {
            public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
                ListNode dummy = new ListNode(0);
                ListNode cur = dummy;
                int carry = 0;
        
                while (l1 != null || l2 != null || carry != 0) {
                    if (l1 != null ) {
                        carry += l1.val;
                        l1 = l1.next;
                    }
        
                    if (l2 != null) {
                        carry += l2.val;
                        l2 = l2.next;
                    }
        
                    cur.next = new ListNode(carry % 10);
                    cur = cur.next;
                    carry = (int)(carry / 10);
        
                }
        
                return dummy.next;
        
        
        
            }
        }
    

    
这道题关键点在于正好可以用逆序进行答案的返回，所以正好可以把进位往next接。cur是需要不断取余数的 而且cur需要把进位进行更新 while里的条件也至关重要 里面的carry l1 l2任意一个不为0或者null的时候 
        都是可以往前进的。int要用整除。
    


        

        
92 反转链表II

        
            class Solution {
                public ListNode reverseBetween(ListNode head, int left, int right) {
                    if (left == 1) {
                        return reverseN(head, right);
                    }
            
                    head.next = reverseBetween(head.next, left - 1, right - 1);
                    return head;
            
                }
            
                ListNode successor = new ListNode(0);
            
                public ListNode reverseN(ListNode head, int n) {
                    if (n == 1) {
                        successor = head.next;
                        return head;
            
                    }
            
                    ListNode last = reverseN(head.next, n - 1);
            
                    head.next.next = head;
                    head.next = successor;
            
                    return last;
            
            
                }
            }
        

        
这道题关键点在于需要熟练反转链表I。先把需要反转的区域调整到left == 1的位置。然后对剩下的右边的链表进行处理 注意处理successor。

142 环形链表II

    
        public class Solution {
            public ListNode detectCycle(ListNode head) {
                ListNode fast = head;
                ListNode slow = head;
        
                while (fast != null && fast.next != null) {
                    fast = fast.next.next;
                    slow = slow.next;
                    if (slow == fast) {
                        break;
                    }
                }
        
                if (fast == null || fast.next == null) {
                    return null;
                }
        
                fast = head;
        
                while (fast != slow) {
                    fast = fast.next;
                    slow = slow.next;
                }
        
                return slow;
                
            }
        }
    

    
先是需要设定一个函数匹配左右括号。然后遇到左边就append，用一个栈，如果是右括号发现能匹配，就把匹配的接出来。匹配不了直接return False。最后返回看看是不是栈为空，空的话答案就能返回True。

这道题关键点在于 设置好odd和even的起始点。 odd.next = even.next; odd = odd.next; 先把指针处理好，再处理节点递进。 while (even != null && even.next != null) 常用模板。 最后用前面一个接上后面一个就可以了。

160. 相交链表

            public class Solution {
                public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
                    ListNode p1 = headA;
                    ListNode p2 = headB;
            
                    while (p1 != p2) {
                        if (p1 != null) {
                            p1 = p1.next;
                        } else {
                            p1 = headB;
                        }
            
                        if (p2 != null) {
                            p2 = p2.next;
                        } else {
                            p2 = headA;
                        }
                    }
            
                    return p1;
                    
                }
            }

这道题关键点在于 p1 p2 要能够等于null。因为有不存在相交节点的case。所以需要用等于null的情况来返回正确的返回值。不能写if (p1.next != null)

142 环形链表II

        public class Solution {
            public ListNode detectCycle(ListNode head) {
                ListNode fast = head;
                ListNode slow = head;
        
                while (fast != null && fast.next != null) {
                    fast = fast.next.next;
                    slow = slow.next;
                    if (slow == fast) {
                        break;
                    }
                }
        
                if (fast == null || fast.next == null) {
                    return null;
                }
        
                fast = head;
        
                while (fast != slow) {
                    fast = fast.next;
                    slow = slow.next;
                }
        
                return slow;
                
            }
        }

先找到环里相交的点。然后此时把fast重新设置回head。注意 fast行走距离2k，slow行走距离k，假设交点距离环的起点距离m，那么head距离环的起点就是k - m。 此刻，如果fast从头开始到环的起点，slow也同步从交点开始行走，那么他们在环的起点可以相遇而且正好走了k-m距离。这个可以用作结论记住。

力扣2215 找出两数组的不同

            class Solution {
                public List> findDifference(int[] nums1, int[] nums2) {
                    List> res = new ArrayList<>();
                    res.add(new ArrayList<>());
                    res.add(new ArrayList<>());
                    Set s1 = new HashSet<>();
                    for (int i = 0; i < nums1.length; i++) {
                        s1.add(nums1[i]);
                    }
                    Set s2 = new HashSet<>();
                    for (int i = 0; i < nums2.length; i++) {
                        s2.add(nums2[i]);
                    }
            
                    for (Integer int1: s1) {
                        if (!s2.contains(int1)) {
                            res.get(0).add(int1);
                        }
                    }
            
                    for (Integer int2: s2) {
                        if (!s1.contains(int2)) {
                            res.get(1).add(int2);
                        }
                    }
            
                    return res;
            
            
                }
            }

看起来简单的哈希题都不要大意，这道题关键点在于 熟练地使用HashSet和List。这里res就用ArrayList是最方便的。为了避免重复元素出现在答案里，被遍历的数据结构就应该是哈希Set。

1 两数之和

            class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        keys = {}
        for i in range(len(nums)):
            if target - nums[i] in keys:
                return [keys[target - nums[i]], i]
            else:
                keys[nums[i]] = i

这道题关键点在于 设置快慢指针。数一数，当fast没法继续前进的时候，slow正好是中间节点。。

128 最长连续序列

            class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        set_num = set(nums)
        length = 0
        res = 0

        for i in range(len(nums)):
            if nums[i] - 1 in set_num:
                continue

            cur = nums[i]
            length = 1

            while cur + 1 in set_num:
                cur = cur + 1
                length = length + 1
            
            res = max(length, res)
        return res

这题如果用O(n) 说实话很难。实际做法是一个摊销分析后的O(n)。（amortized analysis）

        if (len(nums) == 0): return 0
        length = 1
        res = 1
        nums.sort()

        for i in range(1, len(nums)):
            if nums[i] == nums[i - 1]:
                continue
            if nums[i] == nums[i - 1] + 1:
                length += 1
            else:
                
                length = 1
            res = max(res, length)

13 罗马数字转整数

        class Solution {
            public int romanToInt(String s) {
                int preNum = getValue(s.charAt(0));
                int sum = 0;
                for (int i = 1; i < s.length(); i++) {
                    int num = getValue(s.charAt(i));
                    if (preNum < num) {
                        sum -= preNum;
                    } else {
                        sum += preNum;
                    }
                    preNum = num;
        
        
                }
                sum += preNum;
                return sum;
        
            }
        
            private int getValue(char ch) {
                switch(ch) {
                    case 'I': return 1;
                    case 'V': return 5;
                    case 'X': return 10;
                    case 'L': return 50;
                    case 'C': return 100;
                    case 'D': return 500;
                    case 'M': return 1000;
                    default: return 0;
                }
                
            }
        
        
        }

实际不算是哈希题目 但是还可以，姑且作为题目练手，对字符串练习很有用。

142 环形链表II

        public class Solution {
            public ListNode detectCycle(ListNode head) {
                ListNode fast = head;
                ListNode slow = head;
        
                while (fast != null && fast.next != null) {
                    fast = fast.next.next;
                    slow = slow.next;
                    if (slow == fast) {
                        break;
                    }
                }
        
                if (fast == null || fast.next == null) {
                    return null;
                }
        
                fast = head;
        
                while (fast != slow) {
                    fast = fast.next;
                    slow = slow.next;
                }
        
                return slow;
                
            }
        }

先找到环里相交的点。然后此时把fast重新设置回head。注意 fast行走距离2k，slow行走距离k，假设交点距离环的起点距离m，那么head距离环的起点就是k - m。 此刻，如果fast从头开始到环的起点，slow也同步从交点开始行走，那么他们在环的起点可以相遇而且正好走了k-m距离。这个可以用作结论记住。
